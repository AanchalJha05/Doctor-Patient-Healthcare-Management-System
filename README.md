\ 🏥 Doctor-Patient Healthcare Management System



A Django-based web application to manage doctors, patients, and appointments efficiently.



\---



\ 🚀 Features

\- Doctor and Patient registration

\- Appointment booking system

\- Status tracking (Pending, Confirmed, Cancelled)

\- Admin panel for management



\---



\🛠️ Tech Stack

\- Python (Django)

\- SQLite

\- HTML/CSS



\---



\ ⚙️ Setup

```bash

git clone https://github.com/AanchalJha05/Doctor-Patient-Healthcare-Management-System.git

cd Doctor-Patient-Healthcare-Management-System

pip install django

python manage.py migrate

python manage.py runserver
